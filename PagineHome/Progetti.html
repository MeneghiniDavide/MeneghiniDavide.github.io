Progetti

Orario scolastico

Piccolo programma HTML + CSS con la tabella dell'orario scolastico della classe 4^ C INFO e i collegamenti alle pagine HTML dei professori.

Il mondo degli animali

Pagina HTML + CSS riprodotta dalla comsegna del compito contenente un sito riguardante la fauna del mondo.

Mantova

Pagina HTML + CSS riprodotta dalla comsegna del compito e modificato esteticamente secondo indicazioni della consegna con un file CSS

Tavola pitagorica

Pagina HTML + CSS + JS contenente una tavola pitagorica generata con funzioni JavaScript. La tavola pitagorica è 10x10 e all'interno di essa, le caselle pari vengono colorate di rosso.

Lampadina accesa/spenta

Pagina HTML + CSS +JS contenente una stringa che tramite un pulsante cambia testo e una lampadina chre tramite due pultanti può essere accesa o spenta cambiando anche il colore dello sfondo. Inoltre è presente un pulsante che scrive un testo nella console del sito.
